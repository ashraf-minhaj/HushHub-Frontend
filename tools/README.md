## setup environment first

- Install latest version of docker in your system
- Install python3 on your system (should come with your distro BTW)

- run tool - 
    ```
        $ sudo python3 devtool.py <arg> <value>
    ```
- list of args -
    ``` 
        arg          -   value
        help       
        list_images 
        run              
        stop            detach or all
        logs            
        errors          
    ```
